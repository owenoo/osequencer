== OSequencer

A program that converts a list of strings into a list of 4-letter substrings without duplicates.

Steps to run:
1. Clone repository to local machine
2. In terminal navigate to repository clone
3. Run bundle install
3. Enter 'rails console'
4. Invoke DictionaryReader.process
5. Navigate to "app/assets/text/output" and review Sequences.txt and Words.txt
